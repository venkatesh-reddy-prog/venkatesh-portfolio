A **modern, interactive developer portfolio website designed and developed by B. Venkatesh Reddy** using **React**, **Tailwind CSS**, and **JavaScript**.

The portfolio showcases my **projects, technical skills, education, achievements, and coding profiles** through an engaging user interface with smooth animations and responsive design.

---

# 🌐 Live Portfolio

🚀 **Live Website**

https://venkatesh-reddy.vercel.app

This portfolio is **deployed and hosted on Vercel** with automatic deployment connected to my GitHub repository.

Every new push to the repository automatically updates the live website.

• Modern responsive UI design

# ⚙️ Local Development Setup

If you want to run this project locally:

### 1️⃣ Clone the Repository

```
git clone https://github.com/venkatesh-reddy-prog/venkatesh-portfolio.git
cd venkatesh-portfolio
```

---

### 2️⃣ Install Dependencies

```
npm install
```

This installs all required dependencies inside the **node_modules** folder.

---

### 3️⃣ Run Development Server

```
npm start
```

The application will run at:

```
http://localhost:3000
```

The development server supports **hot reload**, so changes appear instantly when files are saved.

# 🐛 Troubleshooting

| Problem              | Solution                           |
| -------------------- | ---------------------------------- |
| npm install fails    | Install Node.js v16 or higher      |
| Port already in use  | Allow React to run on another port |
| Styles not applying  | Verify Tailwind configuration      |
| Missing dependencies | Run `npm install` again            |
